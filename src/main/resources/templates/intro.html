C.愛.랑(씨애랑)
1992년부터 이어져 온 유서 깊은 동아리로써 씨는 Computer 愛는 사랑애, 랑은 순우리말로 모임이라는 뜻으로, 컴퓨터를 사랑하는 사람들이 만든 모임이라는 뜻을 가지고 있습니다. 현재 C.愛.랑은 라떼, 라온, 코어, 태그, 텐서 총 5개의 팀으로 구성되어 있습니다.


수상이력
- 2018 한림대학교 SW융합대학 SW동아리 컨테스트 금상, 해커톤 대상
- 2017 한림대학교 공과대학 학술제 우수 학술 동아리, 학술제 개인 및 단체분야 1위
- 2017 창의·융합 캡스톤디자인 경진대회 금상
- 2017 대기만성 인사만사 캡스톤디자인 경진대회 은상
- 2015 한림대학교 공과대학 학술제 우수동이리, 학술제 단체부문 우수상
- 2015 한림대학교 창업경진대회 대상, 캡스톤 디자인 경진대회 금상
- 2013 강원도 대학생 창업경진대회 우수상, 장려상
- 2012 강원도 대학생 창업 경진대회 최우수상
- 2011 LG Smart TV App idea 부분 동상
라온
씨애랑의 게임 개발 팀입니다. 주로 Unity 3D를 배우며 게임프로젝트 제작을 목표로 합니다. 학습 내용은 C언어와 Unity3D 입니다. 교내 ‘VR/AR 게임제작기초’ 수업과 병행하면 매우 좋습니다.
라떼
씨애랑의 모바일 앱 개발 팀입니다. 현재는 안드로이드 스튜디오를 주로 배우며 여건이 된다면 Swift교육까지 확장해볼 계획입니다. APP 개발을 목표로 하며 학습 내용은 JAVA와 안드로이드 스튜디오 입니다. 교내 ‘모바일 프로그래밍’ 수업과 병행하면 매우 좋습니다.
코어
씨애랑의 아두이노 팀입니다. 주로 아두이노를 배우며 아두이노를 이용한 최종 프로젝트 제작을 목표로 합니다. 학습 내용은 아두이노 및 센서 이론입니다. 교내 ‘임베디드 프로그래밍’수업과 병행하면 매우 좋습니다.
태그
씨애랑의 웹 개발 팀입니다. 크게 프론트엔드와 백엔드로 나누어지고 공통적으로 HTML과 자바스크립트를 배우며 최종 웹 사이트 제작을 목표로 합니다. Front-End 주된 학습 내용은 HTML, JavaScript, CSS 입니다. Back-End는 Spring-Boot를 기반으로 학습합니다.‘웹 프로그래밍, 웹 프레임워크, 웹 서버 구축’ 수업과 병행하면 좋습니다.
텐서
씨애랑의 AI 개발 팀입니다. 머신러닝과 딥러닝을 배우며 AI모델을 이용한 프로젝트 제작을 목표로 합니다. 학습 내용은 Python, 머신러닝, 딥러닝이며 교내 ‘딥러닝 기초’, ‘머신러닝’, ‘자연어 처리’, ‘영상처리와 딥러닝’ 수업과 병행하면 매우 좋습니다.
